## JavaScript-функционал

### nav.js — Навигация
- Добавляет класс `.scrolled` (матовое стекло) после прокрутки на 60px
- Бургер-кнопка открывает/закрывает мобильную шторку
- Закрытие по клику на ссылку или клавише `Escape`
- Блокирует `overflow: hidden` на `body` при открытой шторке

### menu.js — Фильтр меню
- Читает атрибут `data-filter` у табов и `data-cat` у карточек
- Скрывает карточки через атрибут `hidden` с fade-анимацией
- Работает только с `.menu-tabs .btn-tab` (не захватывает кнопки галереи)

### modal.js — Модальные окна
- Клик на карточку меню читает `data-*` атрибуты и заполняет модал
- Закрытие: кнопка ×, клик по оверлею, клавиша `Escape`
- Кнопка «Добавить» внутри модала с состоянием «Добавлено»

### cart.js — Кнопки «Добавить»
- Временно меняет текст кнопки на «✓ Добавлено» на 1.6 секунды
- `stopPropagation()` предотвращает открытие модала при клике на кнопку
- Запускает ken-burns эффект на hero после загрузки страницы

### slider.js — Слайдер отзывов
- Переключение через `translateX` по 100%
- Автопрокрутка каждые 5 секунд
- Управление: стрелки, точки-индикаторы, свайп, клавиши ← →
- Пауза при наведении и когда секция вне viewport (IntersectionObserver)
- Стрелки не перехватывают события когда открыт лайтбокс

### gallery.js — Галерея
- Фильтр по категориям: `data-gfilter` → `data-gcat`
- Лайтбокс: навигация стрелками, свайп, клавиши, клик по фону
- При фильтрации лайтбокс работает только с видимыми элементами

### counter.js — Счётчик
- Срабатывает при появлении секции «О нас» в viewport
- Анимирует числа от 0 до `data-target` с ease-out замедлением
- Суффикс берётся из атрибута `data-suffix`

### reservation.js — Бронирование
- Кастомный счётчик гостей (1–20) с кнопками − и +
- Автоматическая маска телефона: `+7 (XXX) XXX-XX-XX`
- Минимальная дата — сегодня (через `dateInput.min`)
- Имитация отправки с показом сообщения об успехе

### scroll-top.js — «Наверх»
- Кнопка появляется после прокрутки на 400px
- Полоса прогресса чтения вверху страницы
- Плавный скролл через `scrollTo({ behavior: 'smooth' })`

## CSS-архитектура

### Дизайн-токены (`variables.css`)

```css
:root {
  --bg:           #f7f4ef;   /* тёплый кремовый фон            */
  --surface:      #ffffff;   /* карточки, панели               */
  --text:         #13100d;   /* основной текст                 */
  --text-muted:   #8c7b6a;   /* вторичный текст                */
  --accent:       #704030;   /* фирменный коричневый           */
  --accent-light: #f3ece4;   /* светлый тинт для hover         */
  --border:       #e8dfd6;   /* разделители                    */

  --font-display: 'Playfair Display', Georgia, serif;
  --font-body:    'Inter', system-ui, sans-serif;
}
```

### Система кнопок (`buttons.css`)

| Класс | Назначение |
|---|---|
| `.btn .btn-dark` | Основное действие — тёмная заливка |
| `.btn .btn-outline` | Вторичное — контур на светлом фоне |
| `.btn .btn-outline-light` | Контур на тёмном/фотофоне |
| `.btn-tab` | Таб-фильтр с pill-формой |
| `.btn-add` | Кнопка «Добавить» на карточке |
| `.btn-sm` | Уменьшенный размер |

### Важный фикс

Атрибут `[hidden]` переопределяется правилами вроде `display: flex`. В `base.css` добавлен глобальный сброс:

```css
[hidden] { display: none !important; }
```

Без этой строки фильтры меню и галереи не работали бы.

## Адаптивность

Все breakpoints в `responsive.css`:

| Breakpoint | Устройство | Изменения |
|---|---|---|
| `≤ 1200px` | Большой планшет | Галерея: 3 колонки |
| `≤ 1024px` | Планшет | Бургер-меню, features 2×2, about в 1 колонку |
| `≤ 768px` | Планшет портрет | Галерея 2 колонки, слайдер полная ширина |
| `≤ 640px` | Мобильный | Menu 1 колонка, форма в 1 колонку, footer 1 колонка |
| `≤ 480px` | Узкий мобильный | Галерея 1 колонка, компактный hero |

## Как запустить

Никаких зависимостей, сборщиков или серверов не нужно.

```bash
# Просто откройте файл в браузере:
index.html
```

Или через расширение **Live Server** в VS Code для автообновления при изменениях.

## Известные решения

**Карта без API-ключа** — используется OpenStreetMap через `iframe`.

**Шрифты** — загружаются с Google Fonts. При отсутствии интернета браузер использует fallback: `Georgia` для заголовков, `system-ui` для текста.

**Изображения** — все фото загружаются с Unsplash. Атрибут `loading="lazy"` обеспечивает ленивую загрузку.

**Отправка формы** — имитируется через `setTimeout`. Для реальной отправки замените блок в `reservation.js` на `fetch()` к вашему бэкенду.
